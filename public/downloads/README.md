## How to Add Documents

1. Place PDF files directly in this `/public/downloads/` folder
2. Link to them in your code using: `href="/downloads/filename.pdf"`
3. Files will be automatically available at: `https://yoursite.com/downloads/filename.pdf`

## Best Practices

- Use lowercase filenames with hyphens (e.g., `budget-worksheet.pdf`)
- Keep filenames descriptive but concise
- Optimize PDF file sizes before uploading
- Consider adding version numbers for templates that may be updated (e.g., `budget-worksheet-v2.pdf`)
